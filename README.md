Script e dados utilizados para realizar um Etograma de Saguinus midas no Zoológico de Itatiba, para a disciplina NE466 (Introdução ao comportamento animal e filogenia) da UNICAMP (Universidade Estadual de Campinas).

Dados são:
# Dados de comportamentos em 50 intervalos de horários (T1, T2...)
# Horário dividido em 5 minutos de intervalo e 1 minuto de varredura
# 8 classes de comportamento
# 100 atos comportamentais 

# Observação das 9h:35 às 15h:20

# 7 indivíduos de S. midas no mesmo recinto, sem diferenças marcantes entre indivíduos (macho x fêmea; jovem x adulto)
